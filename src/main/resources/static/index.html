Шахматная доска с серверным взаимодействием

Вы играете за белых, сервер играет за черных

Текущий ход: Белые (ваш ход)

Как использовать:

• Кликните на клетку, чтобы разместить белую фишку (ваш ход)

• После вашего хода автоматически отправляется запрос на сервер

• Сервер отвечает ходом черных, который отображается на доске

• Измените размер доски с помощью поля ввода

• Используйте кнопку "Очистить доску" для сброса всех фишек
